National Parks Res

Reservation system built for reserving camp site at a National Park. User sorts through each park and their campgrounds/campsites with various information. User then has the ability to see if the campsite they wish to reserve is available for their time frame and then can book with total cost of reservation. If user has special requirements for their stay the advanced search function shows users all campsites with extra amenities.

On individual Park’s page there is more detailed information about the park, including the upcoming 5 day forecast. Echo Flash

Flash card web application built for user to study. As an unlogged in user you can search through premade decks to study and quiz themselves on the decks. As a register/logged in user you can create decks. Create, edit and delete individual card, while also having the ability to add and remove cards from decks. As a logged in user who did not create the card they have the ability to add/remove cards to/from their own decks.
